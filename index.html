Présentation des plugins Intro.js et Ip-Grid.js

Ip-Grid.js

Le plugin Ip-Grid permet de générer une feuille de calcul dans une page web, dans le même principe que Google Sheet.

Pour cela, il faut créér la structure html et initialiser cette dernière en JavaScript.

Intro.js

Le plugin Intro.js permet de faire une présentation pas à pas.

Par exemple, pour un nouvel utilisateur, il vous sera possible de montrer l'utilisation d'une de vos page web, ou d'un de vos API.

Pour cela, il vous suffit de définir l'ordre des différentes étapes. Celles ci seront alors mises en avant, accompagné d'un texte explicatif.
